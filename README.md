## My To-Do List App

A React Native mobile app built with Expo.

## Features
- Add tasks
- Mark tasks as complete
- Delete tasks
- Data saved locally with AsyncStorage

## How to Run
1. Install Node.js
2. Run: npx create-expo-app TodoApp
3. Replace app/(tabs)/index.tsx with the index.tsx file
4. Run: npm install @react-native-async-storage/async-storage
5. Run: npx expo start
6. Scan QR code with Expo Go app
